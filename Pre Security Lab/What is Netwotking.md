### Key Concepts Covered
* **What is Networking?** Networking is the practice of linking two or more computing devices together to share data, resources, and services.
* **The Internet:** The Internet is a global network of computer networks. It relies on a standardized set of rules (**Protocols**) to facilitate communication between devices globally.
* **Device Identification:** Devices on a network are uniquely identified using several methods:
    * **IP Address:** A logical address used for routing traffic across different networks (e.g., $192.168.1.1$).
    * **MAC Address:** A physical, unique hardware identifier assigned to a network interface card (NIC) by the manufacturer.
* **Routers and Switches:** These are key devices that direct network traffic. **Routers** handle traffic between different networks, and **Switches** handle traffic within a single network.
* **Ping (ICMP):** Learned about the **I**nternet **C**ontrol **M**essage **P**rotocol and the `ping` command, which is used to test the reachability and measure the round-trip time (latency) to another host on an IP network.
